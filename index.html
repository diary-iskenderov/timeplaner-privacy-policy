Privacy Policy for Планер+ (Timeplaner)

Last updated: March 19, 2026

Эта Политика конфиденциальности объясняет, какие данные приложение Планер+ (Timeplaner) собирает, как использует, хранит и защищает их.

1. Какие данные мы собираем

  • Данные аккаунта: email, имя профиля и UID при входе через Google / Firebase Authentication.
  • Пользовательские данные: задачи, группы, статусы задач, друзья, история выполнения и связанные настройки планирования.
  • Технические данные: служебные данные Firebase/Google, необходимые для авторизации, синхронизации и работы приложения.
  • Локальные настройки: тема, цветовые настройки, расположение блоков рабочего стола (хранятся на устройстве).

2. Как мы используем данные

  • Для входа в аккаунт и защиты доступа.
  • Для синхронизации задач и групп между устройствами.
  • Для отображения аналитики и прогресса пользователя.
  • Для улучшения стабильности и качества приложения.

3. Сторонние сервисы

Приложение использует:

  • Firebase Authentication
  • Cloud Firestore
  • Google Sign-In
  • Firebase Analytics (если включено в сборке)

Политика Google: https://policies.google.com/privacy

4. Передача данных

Мы не продаем персональные данные. Передача данных возможна только сервисам, необходимым для работы функций приложения (авторизация, облачная синхронизация, аналитика).

5. Срок хранения данных

Данные хранятся, пока это необходимо для работы приложения и аккаунта пользователя, либо до запроса на удаление.

6. Удаление аккаунта и данных

Разработчик: Diary Iskenderov
Приложение: Планер+ (Timeplaner)

Как удалить данные

  1. В приложении: Профиль → Удалить аккаунт и данные.
  2. По email: diary.iskenderov@gmail.com
  3. Через Telegram: @diary_iskenderov

При удалении удаляются данные аккаунта и связанные облачные данные (задачи, группы, история, друзья), за исключением данных, которые должны храниться по закону.

7. Безопасность

Мы применяем разумные технические меры защиты. Однако ни один способ хранения и передачи данных не гарантирует абсолютную безопасность.

8. Дети

Приложение не предназначено для детей младше возраста, установленного законодательством, без согласия родителей.

9. Права пользователя

В зависимости от страны вы можете запросить доступ, исправление или удаление данных. Для запросов используйте контакты выше.

10. Изменения политики

Мы можем обновлять эту Политику конфиденциальности. Актуальная версия всегда публикуется на этой странице.
